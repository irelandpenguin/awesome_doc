Introduction

LiteIDE is a simple, open source, cross-platform IDE.

Base Features:

  • Mime type basis system
  • System environment manage
  • Build system manage
  • Debug system simple and open
  • Kate syntax and style scheme
  • WordApi complete helper

Golang Support:

  • GOPATH Project
  • Go Playground
  • Golang ast view
  • Godoc browser
  • Gocode helper
  • Project wizard
  • Project build
  • Source build

System

  • Windows
  • Linux
  • MacOSX
