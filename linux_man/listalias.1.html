NAME

listalias - 列出用户和系统别名

总揽

listalias [ -s | -u ] [ 正则表达式]

描述

Listalias 按用户及系统别名每个输出一行。每行具有下列格式： 
   
   <别名>     <地址>  (<注解>)
如果采用了正则表达式选项，则只有与指定表达式匹配的别名才被列出， 如果未采用正则表达式，所有别名将被列出。

在两种方式下，命令的输出都被排序。

选项

-s
只输出系统别名。  -s 和 -u 两个选项为互斥关系，即两个选项只能选一个。
-u
只输出用户别名  -s 和 -u 两个选项为互斥关系。
