NAME

rpmdeps - 生成 RPM 软件包依赖关系

SYNOPSIS

rpmdeps {-P|--provides} {-R|--requires} FILE ...

DESCRIPTION

rpmdeps 根据 FILE 参数集合，生成软件包依赖关系。FILE 参数中的每个都进行搜索，查找 Elf32/Elf64，脚本解释器，以及每个脚本的依赖性关系，将依赖性关系输出到标准输出。
