NAME

dircolors - 设置 ls 的颜色

总览 (SYNOPSIS)

dircolors [OPTION]... [FILE]

描述 (DESCRIPTION)

输出 用于 设置 LS_COLORS 环境变量 的 命令.

确定输出格式:

-b, --sh, --bourne-shell
输出 用于 设置 LS_COLORS 的 Bourne shell 命令
-c, --csh, --c-shell
输出 用于 设置 LS_COLORS 的 C shell 命令
-p, --print-database
输出 缺省命令
--help
显示 帮助信息, 然后 结束
--version
显示 版本信息, 然后 结束

如果 指定了 文件 FILE, dircolors 读取 该文件 的 内容 以 决定 什么 颜色 对应 什么 文件类型 和 文件扩展名, 否则 就 使用 预编译 的 数据. 欲知 格式 细节, 请 运行 命令 `dircolors --print-database'.
