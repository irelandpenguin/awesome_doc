NAME

pgtksh -- PostgreSQL Tcl/Tk shell 客户端

SYNOPSIS

pgtksh [filename [argument...]]

DESCRIPTION 描述

pgtksh 是一个带有 PostgreSQL 数据库访问函数扩展的 Tcl/Tk shell 接口。（实际上，它是装载了 libpgtcl 的wish。） 和普通的 Tcl/Tk shell wish 类似，命令行的第一个参数是一个脚本文件， 任何剩余的参数都传递给该脚本。特殊的选项可以由 X Window 系统库来处理。 如果没有命名脚本的名字，那么该 shell 是交互的。


 一个带有 PostgreSQL 函数的纯 Tcl shell 是 pgtclsh(1)。

SEE ALSO 参见

pgtclsh(1), （描述了 libpgtcl）, tclsh(1), wish(1)
