NAME

logname - 显示用户登录名

总览 (SYNOPSIS)

logname [OPTION]...

描述 (DESCRIPTION)

显示 当前用户 的 名字.

--help
显示 帮助信息, 然后 结束.
--version
显示 版本信息, 然后 结束.

报告 BUGS

发现 的 错误 送往 <bug-sh-utils@gnu.org>.

另见 (SEE ALSO)

logname 的 完整 文档 以 Texinfo 手册 的 形式 维护. 如果 正确 安装 了 info 和 logname 程序, 使用 命令
info logname

能够 访问到 完整 的 手册.
