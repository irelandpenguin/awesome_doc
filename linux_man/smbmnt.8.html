NAME 名字

smbmnt - 装载 SMB 文件系统的协助工具。

总览

smbmnt mount-point [ -s share ] [ -r ] [ -u uid ] [ -g gid ] [ -f mask ] [ -d mask ]

描述

smbmnt 用于协助 smbmount 程序完成实际装载 SMB 共享资源。 smbmnt 必须以 root 用 setuid 来安装，这样普通用户才能 装载他们的 SMB 共享资源。它在装载点和装载目录上检查 用户是否有写权限。

smbmnt 通常由 smbmount 来调用。用户不能直接调用它。

-r
以只读方式装载文件系统

-u uid
指定要装载文件者的用户标识符。

-g gid
指定要装载文件者的组标识符。

-f mask
声明用八进制数表示的文件掩码。

-d mask
声明用八进制数表示的文件掩码。
