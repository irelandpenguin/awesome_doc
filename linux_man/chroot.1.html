NAME

chroot - 以 特定 根 目录 运行 命令 或者 交互式 shell

总览 (SYNOPSIS)

chroot [OPTION] NEWROOT [COMMAND...]
chroot OPTION

描述 (DESCRIPTION)

以 NEWROOT 为 根 目录 运行 COMMAND.

--help
显示 帮助 并且 退出
--version
显示 版本 信息 并且 退出

如果 没有 指定 命令, 运行 ``${SHELL} -i'' (默认: /bin/sh).
