NAME

iptables-restore - 恢复 IP Tables

总览 SYNOPSIS

iptables-restore [-c] [-n]

描述 DESCRIPTION

iptables-restore 用来从 STDIN 给出的数据中恢复 IP Tables。 使用 shell 的 I/O 重定向功能来从文件中获取数据

-c, --counters
恢复所有报文和字节计数的值
-n, --noflush
不刷新表中从前的内容。如果没有指定，
iptables-restore 会刷新 (删除) 相应 IP Tables 中所有从前的内容
