NAME

bzero - 向字符串写入零

总览 (SYNOPSIS)

#include <string.h>

void bzero(void *s, size_t n);

描述 (DESCRIPTION)

bzero() 函数 把 字符串 s 的 前 n 个 字节 置成 零.

返回值 (RETURN VALUE)

bzero() 函数 无 返回值.

遵循 (CONFORMING TO)

4.3BSD. 建议 不要 用 这个 函数 -- 请 在 新的 程序中 用 memset 函数.
