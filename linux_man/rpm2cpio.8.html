NAME

rpm2cpio - 从 RPM 软件包中提取 cpio 归档

SYNOPSIS

rpm2cpio [filename]

DESCRIPTION

rpm2cpio 将指定的一个 .rpm 文件转换为一个 cpio 文档，输出到标准输出。如果给出了 `-' 参数，那么将从标准输入读取 rpm 文件。


rpm2cpio rpm-1.1-1.i386.rpm
rpm2cpio - < glint-1.0-1.i386.rpm
