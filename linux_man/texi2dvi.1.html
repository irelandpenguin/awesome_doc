NAME

texi2dvi - 打印 Texinfo 文档

SYNOPSIS 总览

texi2dvi [OPTION]... FILE...

DESCRIPTION 描述

依次从 Tex 系统中运行每个 Texinfo 或者 LaTex 文件 FILE，直到解决了所有的交叉引用，建立所有的索引。每个包含 FILE 的目录都将被搜索，查找被包含的文件。FILE 的后缀被用来判断它的语言 (LaTex 还是 Texinfo)。

需要的时候，在运行 Tex 系统之前，使用 makeinfo 来进行 Texinfo 宏扩展。

操作模式：

-b, --batch
非交互式
-c, --clean
删除所有的辅助文件 (auxiliary)
-D, --debug
打开 shell 调试 (设置 -x)
-h, --help
显示此帮助，然后退出
-o, --output=OFILE
将输出保存于 OFILE (隐含着 --clean)；这种情况下只能指定一个输入文件 FILE
-q, --quiet
不显示输出，除非发生错误 (隐含着 --batch)
-s, --silent
与 --quiet 相同
-v, --version
显示版本信息，然后退出
-V, --verbose
报告每一步行为

调整 TeX：

-@
使用 @input 而不是 \input；只对应预先加载的 Texinfo
-e, -E, --expand
强制使用 makeinfo 进行宏扩展
-I DIR
搜索 DIR，查找 Texinfo 文件
-l, --language=LANG
指定文件 FILE 的语言 LANG (值为 LaTex 或 Texinfo)
-p, --pdf
使用 pdftex 或者 pdflatex 来处理
-t, --command=CMD
在输入文件的拷贝中插入 CMD；
或 --texinfo=CMD
累积多个值 (?)

环境变量 BIBTEX, LATEX (或 PDFLATEX), MAKEINDEX, MAKEINFO, TEX (或 PDFTEX), 以及 TEXINDEX 的值将用来运行对应的命令，如果设置了它们的话。任何 CMD 字串都被添加到 Texinfo 输入文件的 @setfilename 之后，或者 LaTex 输入文件的第一行。

REPORTING BUGS 报告错误

将错误报告发送到 bug-texinfo@gnu.org，一般的问题和讨论则发送到 help-texinfo@gnu.org。 Texinfo 主页：http://www.gnu.org/software/texinfo/
