NAME

sum - 计算文件的校验和,以及文件占用的块数

总览 (SYNOPSIS)

../src/sum [OPTION]... [FILE]...

描述 (DESCRIPTION)

显示 每个 文件 FILE 的 校验和, 以及 他们 占用的 块数.

-r
取代 -s, 使用 BSD sum 算法, 块 大小 为 1K
-s, --sysv
使用 System V sum 算法, 块 大小 为 512 字节
--help
显示 帮助信息, 然后 结束
--version
显示 版本信息, 然后 结束

如果 没有 指定 FILE, 或 FILE 是 -, 就从 标准输入 读取 数据.
