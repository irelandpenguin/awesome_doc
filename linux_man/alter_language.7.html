NAME

ALTER LANGUAGE - 修改一个过程语言的定义

SYNOPSIS

ALTER LANGUAGE name RENAME TO newname

DESCRIPTION 描述

ALTER LANGUAGE 修改一门语言的定义。 目前唯一的功能就是重命名语言。只有超级用户可以重命名一门语言。

PARAMETERS 参数

name

 一门语言的名字
newname

 该语言的新名字

COMPATIBILITY 兼容性


 SQL 标准里没有 ALTER LANGUAGE 语句。

SEE ALSO 参见

CREATE LANGUAGE [create_language(7)], DROP LANGUAGE [drop_language(l)]
