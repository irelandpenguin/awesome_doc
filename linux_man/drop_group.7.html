NAME

DROP GROUP - 删除一个用户组

SYNOPSIS

DROP GROUP name

DESCRIPTION 描述

DROP GROUP 从数据库中删除指定的组。组中的用户不被删除。 组中的用户不被删除。

PARAMETERS 参数

name

 现存组名。

EXAMPLES 例子


 删除一个组： 

DROP GROUP staff;

COMPATIBILITY 兼容性


 SQL 标准里没有 DROP GROUP。

SEE ALSO 参见

ALTER GROUP [alter_group(7)], CREATE GROUP [create_group(l)]
