NAME

ALTER TRIGGER - 修改一个触发器的定义

SYNOPSIS

ALTER TRIGGER name ON table RENAME TO newname

DESCRIPTION 描述

ALTER TRIGGER 改变一个现有触发器的属性。 RENAME 修改一个给出地触发器地名称， 而不用改变触发器的定义。


 你必需拥有该触发器作用的表才能改变其属性。

PARAMETERS 参数

name

 现有需要修改的触发器的名称。
table

 该触发器作用的表的名字。
newname

 现有触发器的新名字。

EXAMPLES 例子


 重新命名一个现有触发器： 

ALTER TRIGGER emp_stamp ON emp RENAME TO emp_track_chgs;

COMPATIBILITY 兼容性

ALTER TRIGGER 是 PostgreSQL 对 SQL 标准的扩展。
