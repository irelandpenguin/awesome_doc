NAME

DROP RULE - 删除一个重写规则

SYNOPSIS

DROP RULE name ON relation [ CASCADE | RESTRICT ]

DESCRIPTION 描述

DROP RULE 删除一个规则。

PARAMETERS 参数

name

 要删除的现存的规则。
relation

 该规则应用的关系名字（可以有模式修饰）。
CASCADE

 自动删除依赖于此规则的对象。
RESTRICT

 如果有任何依赖对象，则拒绝删除此规则。这个是缺省。

EXAMPLES 例子


 删除重写规则 newrule： 

DROP RULE newrule ON mytable;

COMPATIBILITY 兼容性


 在 SQL 标准中没有DROP RULE。
