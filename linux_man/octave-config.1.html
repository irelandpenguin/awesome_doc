NAME

octave-config - 检索 GNU Octave 组件和库信息

SYNOPSIS 总览

octave-config [--m-site-dir] [--oct-site-dir] [-v|--version]  [-h|-?|--help]

DESCRIPTION 描述

octave-config 是用于为 octave(1) 获取 和 文件的目录信息的工具。

OPTIONS 选项

octave-config 接受下列选项：
--m-site-dir
显示本地的或 site-specific 的，.m 脚本文件所在的主目录。
--oct-site-dir
显示本地的或 site-specific 的，.oct 动态链接库所在的主目录。 -v|--version 显示 octave(1) 的版本信息。
-h|-?|--help
显示 octave-config 的帮助页面。
