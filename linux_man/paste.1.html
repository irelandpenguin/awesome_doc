NAME

paste - 合并文件各行

总览 (SYNOPSIS)

../src/paste [OPTION]... [FILE]...

描述 (DESCRIPTION)

连续 依次 从 各个 文件 FILE 中 读取 一行 然后 合并成 新行, 中间 用 TAB 隔开, 新行 写到 标准输出. 如果 文件 FILE 不存在, 或者 FILE 是 '-', 就从 标准输入 读取 数据.

-d, --delimiters=LIST
循环使用 LIST 中 的 字符 取代 TAB
-s, --serial
一次 粘贴 一个 文件, 而不是 并行 粘贴
--help
显示 帮助信息, 然后 结束
--version
显示 版本信息, 然后 结束
