NAME

DROP CONVERSION - 删除一个用户定义的编码转换

SYNOPSIS

DROP CONVERSION name [ CASCADE | RESTRICT ]

DESCRIPTION 描述

DROP CONVERSION 删除一个以前定义的编码转换。 要想删除一个转换，你必须拥有该转换。

PARAMETERS 参数

name

 编码转换的名字。这个名字可以用模式修饰。
CASCADE
RESTRICT

 这些键字没有作用，因为编码转换上没有依赖关系。

EXAMPLES 例子


 删除一个叫做 myname 的编码转换： 

DROP CONVERSION myname;

COMPATIBILITY 兼容性


 SQL 标准里没有 DROP CONVERSION。

SEE ALSO 参见

ALTER CONVERSION [alter_conversion(7)], CREATE CONVERSION [create_conversion(l)]
